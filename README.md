# 🎯 SEL for ALL 최종 산출물 제출


## 📘 개요
SEL for all 프로젝트의 moodmeter로 공감하기 최종 산출물 완성본 
본 프로젝트는 특수교육 대상학생을 포함함 학생용 SEL 웹앱으로, 사회정서학습의 시작인 자신의 감정을 알아차리고 친구의 마음을 읽어 서로 감정온도를 맞추는 것을 시각화하고 시뮬레이션한 실습형 포털입니다.

- 🔒 개인정보처리방침 및 사용약관 페이지 포함  
- 🏫 소속학교 및 제작자 이름 표기 완료

- 🌐 프론트엔드 배포 URL 정상 작동 확인

## 📖 사용 방법
1. 사이트에 접속합니다. 
2. 나의 감정, 친구 감정을 알고 온도를 맞춰보는 활동까지 진행합니다.  
3. 각 감정을 좀 더 익히기 위해서는 각 감정알기 아래에 보조적 설명을 익힙니다.
4. 즐거운 무드미터 학습을 위해 보너스 게임을 진행합니다.
5. (개발중) 현재 학생의 감정 추이를 데이터로 모집하는 기능을 개발 중에 있습니다.

## 🌟 프로젝트 의의
이 웹앱은 학생의 모호한 감정을 무드미터로 실시간으로 바뀌는 표정과 에너지바, 쾌적함바의 이동을 통해 익힐 수 있습니다.
감정의 색과 감정을 익힌 후에는 내 감정이 어떤지 알아차린 다음 상대방의 감정과 나의 감정을 맞춰보며 거리를 두어야할 때와 나의 감정을 알려야할 때, 상대방의 감정에 공감하는 방법에 대한 솔루션을 제공합니다.
무드미터의 색과 감정이 모호하지만 게임을 통해 익숙해지도록 게이미피케이션 요소로 감정 학습을 하도록 하였으며 UDL을 적용하여 고도할 예정애 있습니다 (게임단계, 속도, 유저 참여방식, 대비 조절, 감정 단어 및 이모지 변경 옵션 제공 등)
